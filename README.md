# Financial Statement Engine
An AI-assisted financial statement production engine designed to automate the preparation of audited financial statements.

## Objectives
- Trial Balance ingestion
- Mapping automation
- Financial statement generation
- Note disclosure generation
- Audit support schedules
- Singapore SFRS compliance
